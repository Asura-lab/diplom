\textbf{\large Хураангуй}

\sloppy

Энэхүү судалгааны ажлаар машин сургалтын ансамбль аргуудыг ашиглан валютын зах зээл (Forex) дээрх EUR/USD хослолын үнийн чиг хандлагыг таамаглах, автомат арилжааны дохио үүсгэх систем болон гар утасны аппликейшн хөгжүүлсэн.

Системийн цөм нь LightGBM, XGBoost, CatBoost гэсэн гурван градиент нэмэгдүүлсэн шийдвэрийн модны (GBDT) ансамбль загвар юм. Загваруудыг 2015--2022 оны $\sim$2.97 сая лаа (candle) ашиглан сургаж, 2023 оны 371,000 лаа (candle)-аар баталгаажуулж, 2024 оны 371,000 лаа (candle)-д тестэлсэн. Урагш гүйлгэн баталгаажуулах (walk-forward validation) аргачлалыг хэрэглэж хэт тааруулалт (overfitting)-ийн эрсдэлийг бууруулсан. 6 хугацааны интервалаас (M1--H4) нийт 48 техник индикатор тооцоолж, олон хугацааны хүрээний дүн шинжилгээг хэрэгжүүлсэн. Тохируулга (Calibration)-ийг сүлжээнээс гадуурх (offline) үнэлгээнд ашиглаж, бодит ажиллах орчин (production runtime) дээр ансамблийн шууд магадлалд суурилсан итгэлцэл (confidence) бодлого хэрэглэсэн.

2025 оны бэктест (backtest) дүн: 10,000\$ анхны хөрөнгөөс 14,161.20\$ болж \textbf{+41.61\%} өгөөж, \textbf{Ашгийн хүчин зүйл (Profit Factor) 2.46}, \textbf{Шарпын харьцаа (Sharpe Ratio) 9.64}, \textbf{хамгийн их уналт 3.93\%} хүрсэн. React Native технологи дээр суурилсан ``Predictrix'' гар утасны (mobile) аппликейшн нь бодит цагийн ханш, машин сургалт (machine learning)-ын дохио, эдийн засгийн мэдээ, хэрэглэгчийн баталгаажуулалт зэрэг бүрэн функцийг хэрэгжүүлсэн. Нэвтрэлтийн аюулгүй байдлын хүрээнд богино хугацаат JWT токен (token) (олгогч/хүлээн авагч (issuer/audience) баталгаажуулалттай), арын процессын давхардлыг бууруулах тархсан түгжээ (distributed lock), болон Fly орчны бодит хэрэглээний (production) байршуулалтын үйл ажиллагааны (operational) шалгалтыг системийн хэрэгжилтэд тусгасан.

\begin{center}
\textbf{\large Abstract}
\end{center}

\begin{english}
This research develops an automated trading signal generation system and mobile application for predicting EUR/USD price movements in the Forex market using machine learning ensemble methods.

The core of the system is an ensemble of three Gradient Boosted Decision Tree (GBDT) models: LightGBM (496 trees), XGBoost ($\sim$400 trees), and CatBoost (499 trees). The models were trained on $\sim$2.97 million bars from 2015--2022, validated on 371,000 bars from 2023, and tested on 371,000 bars from 2024. Walk-forward validation methodology was employed to mitigate overfitting risks. A total of 48 technical features were computed from 6 timeframes (M1, M5, M15, M30, H1, H4), implementing multi-timeframe analysis. Calibration was retained for offline evaluation, while production runtime uses direct ensemble probability confidence for stability.

Backtest results for 2025: starting from \$10,000 initial capital, the system achieved \$14,161.20 final balance (\textbf{+41.61\%} return), \textbf{Profit Factor 2.46}, \textbf{Sharpe Ratio 9.64}, and \textbf{Maximum Drawdown 3.93\%}. The ``Predictrix'' mobile application, built with React Native, provides real-time exchange rates, machine learning signals, economic news, and user authentication. System engineering contributions include short-lived JWT with issuer/audience validation, distributed runtime locks for background jobs, and production deployment verification on Fly.

\textit{Keywords: machine learning, ensemble model, foreign exchange market, GBDT, XGBoost, LightGBM, CatBoost, technical analysis, trading signals, React Native, backtesting}
\end{english}
