\textbf{\large Хураангуй}
\end{center}

\sloppy

Энэхүү судалгааны ажлаар машин сургалтын (Machine Learning) ансамбль аргуудыг ашиглан санхүүгийн шуугиан ихтэй (high-noise) цаг хугацааны цувааны өгөгдөл дээр таамаглал хийх, улмаар түүнийг хэрэглэгчид бодит хугацаанд дамжуулах өгөгдөл боловсруулах урсгал (Data pipeline) болон харагдах хэсэг (frontend) бүхий мобайл аппликейшн хөгжүүлсэн.

Системийн цөм нь LightGBM, XGBoost, CatBoost гэсэн гурван градиент нэмэгдүүлсэн шийдвэрийн модны (GBDT) алгоритмуудаас бүрдэх санал өгөх механизмтай (Voting Classifier) ансамбль загвар юм. Загваруудыг 2015--2022 оны $\sim$2.97 сая өгөгдлийн цэг (өгөгдлийн түүвэр)-ийг ашиглан сургаж, 2023 оны 371,000 цэг дээр баталгаажуулж, 2024 оны 371,000 цэг дээр тестэлсэн. Загварын хэт нийцэл (хэт нийцэл (overfitting)) үүсэх эрсдэлийг бууруулах зорилгоор олон уулзвар бүхий цагийн дараалалтай баталгаажуулалт (walk-forward validation) хэрэглэсэн. 6 хугацааны интервалаас нийт 48 шинж чанар (feature) боловсруулж сургалтад ашигласан. Загварын магадлалын тохируулгыг (probability calibration) сүлжээнээс гадуурх үнэлгээнд ашигласан бөгөөд бодит ажиллах орчинд (production runtime) ансамблийн шууд магадлалд суурилсан итгэлцлийн хязгаар (confidence threshold) хэрэглэсэн.

Программ хангамжийн туршилтын (system testing) үр дүнд систем нь алгоритмын өндөр нарийвчлалтай ажиллаж, таамаглалын алдааны хэлбэлзэл (алдааны хэлбэлзэл (drawdown)) 3.93\% хүрсэн байна. React Native технологи дээр суурилсан ``Predictrix'' гар утасны харагдах хэсэг (frontend) нь бодит цагийн мэдээлэлгүүд, таамаглалын үр дүн, хэрэглэгчийн баталгаажуулалт зэргийг харуулсан. Серверийн аюулгүй байдлын хүрээнд богино хугацаат JWT токен, арын процессын давхардлыг бууруулах тархсан түгжээ (distributed lock), болон үүлэн (Cloud) орчны байршуулалтыг (deployment) системийн хэрэгжилтэд амжилттай тусгасан.

\begin{center}
\textbf{\large Abstract}
\end{center}

\begin{english}
This research develops an End-to-End machine learning system, which uses ensemble methods for classifying and predicting high-noise, financial time-series data, and integrating a data pipeline with a mobile frontend application to provide real-time results. 

The core of the system is an ensemble voting classifier composed of three Gradient Boosted Decision Tree (GBDT) models: LightGBM, XGBoost, and CatBoost. The models were trained on $\sim$2.97 million data points from 2015--2022, validated on 371,000 points from 2023, and tested on 371,000 points from 2024. A walk-forward validation methodology was implemented to mitigate хэт нийцэл (overfitting) risks. Feature engineering produced 48 distinct features spanning across 6 time intervals (M1--H4). Probability calibration was retained for offline evaluation, while production runtime utilizes direct ensemble probability confidence thresholds for stability.

The system evaluation demonstrates algorithmic robustness with a maximum performance variation error of 3.93\%. The ``Predictrix'' mobile frontend application, developed with React Native, integrates real-time data flows, predictive results, and secure user authentication. System engineering contributions include short-lived JWT instances for access security, distributed locks for background data processes, and a robust production deployment architecture utilizing Fly.io.

\vspace{3mm}
\textit{Keywords: machine learning, ensemble models, time-series data forecasting, feature engineering, data pipelines, GBDT, XGBoost, LightGBM, CatBoost, React Native, frontend application, algorithm testing}
\end{english}
