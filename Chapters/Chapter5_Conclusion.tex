\chapter{ДҮГНЭЛТ}

\section{Судалгааны үр дүнгийн нэгтгэл}

Энэхүү дипломын ажлаар машин сургалтын V10 Ensemble аргыг ашиглан EUR/USD валютын хосын BUY дохио таамаглах системийг амжилттай хөгжүүлсэн. Судалгааны явцад XGBoost×3, LightGBM×2, CatBoost×2 гэсэн долоон загварыг нэгтгэсэн V10 Ensemble арга нь дангаар ажиллах загвараас маш их давуу, 96.9\% нарийвчлалтай үр дүн өгч байгааг баталсан.

\subsection{Техникийн шийдэл}

Загварын сургалтад ашигласан гол арга хэрэгслүүд:

\begin{enumerate}
    \item \textbf{V10 Ensemble загвар:} 7 загварын нэгдэл:
    \begin{itemize}
        \item XGBoost × 3 - Гурван өөр hyperparameter-тэй
        \item LightGBM × 2 - Histogram-based, санах ой хэмнэлттэй
        \item CatBoost × 2 - Ordered boosting, categorical feature сайн
    \end{itemize}
    
    \item \textbf{Agreement Bonus System:} Загваруудын зөвшилцөлд суурилсан нэмэлт оноо:
    \begin{itemize}
        \item 7/7 загвар ижил: +7\% bonus
        \item 6/7 загвар ижил: +4\% bonus
        \item 5/7 загвар ижил: +2\% bonus
    \end{itemize}
    
    \item \textbf{32 техникийн индикатор:} 
    \begin{itemize}
        \item Trend индикаторууд (SMA, EMA, MACD, ADX)
        \item Momentum индикаторууд (RSI, Stochastic, CCI, Williams \%R)
        \item Volatility индикаторууд (ATR, Bollinger Bands)
        \item Time features (London/NY session, overlap)
    \end{itemize}
    
    \item \textbf{Итгэлцлийн шүүлтүүр:} 85\%+ confidence-тэй BUY сигнал = 96.9\% accuracy
    
    \item \textbf{Dynamic SL/TP:} ATR дээр суурилсан Stop Loss, Take Profit тооцоолол
    
    \item \textbf{Entry/SL/TP гаралт:} Тодорхой entry price, stop loss, take profit утгууд
\end{enumerate}

\subsection{Системийн бүрэлдэхүүн}

Бүрэн ажиллагаатай систем хөгжүүлсэн:

\begin{enumerate}
    \item \textbf{Backend API:} Flask + Waitress WSGI сервер
    \item \textbf{V10 ML Pipeline:} Өгөгдөл татах → 32 индикатор тооцох → 7 загварын Ensemble таамаглал
    \item \textbf{Mobile App:} React Native + Expo мобайл аппликейшн
    \item \textbf{Database:} MongoDB (хэрэглэгчийн мэдээлэл, дохионууд)
    \item \textbf{Data Source:} Twelve Data API (бодит цагийн өгөгдөл)
\end{enumerate}

\section{Төгсгөлийн үг}

Энэхүү дипломын ажлаар V10 Ensemble загварыг (XGBoost×3, LightGBM×2, CatBoost×2) ашиглан EUR/USD валютын хосын BUY дохио таамаглах системийг амжилттай хөгжүүлсэн. 

85\%+ итгэлцэлтэй BUY сигнал нь backtest дээр \textbf{96.9\% accuracy}, \textbf{52.2 profit factor} үзүүлж, ~55 өдрийн туршид \textbf{+1,844 pip} ашиг олсон. Энэ нь практикт маш сайн хэрэглэгдэх боломжтой түвшин юм.

7 загварын нэгдэл ба Agreement Bonus системийн хослол нь итгэлцүүрийг нэмэгдүүлж, алдааг маш бага түвшинд хүргэсэн (зөвхөн 3.1\%). Машин сургалтаар санхүүгийн зах зээлийг төгс таамаглах боломжгүй боловч, статистик давуу талтай арилжааны шийдвэр дэмжлэгийн систем бүтээх бүрэн боломжтой гэдгийг энэхүү судалгаа харуулж байна. 

Profit Factor 52.2 нь мэргэжлийн арилжаачдын хүлээн зөвшөөрдөг маш өндөр үр дүн бөгөөд цаашдын хөгжүүлэлтэд бат суурь болно.
