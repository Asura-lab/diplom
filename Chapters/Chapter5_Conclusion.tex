\chapter{ДҮГНЭЛТ}

\section{Судалгааны үр дүнгийн нэгтгэл}

Энэхүү дипломын ажлаар машин сургалтын Ensemble аргыг ашиглан EUR/USD валютын хосын BUY дохио таамаглах системийг амжилттай хөгжүүлсэн.

\subsection{Техникийн үр дүн}

\begin{enumerate}
    \item \textbf{Ensemble загвар:} XGBoost, LightGBM, Random Forest гурван загварыг нэгтгэсэн:
    \begin{itemize}
        \item XGBoost (40\% жин) - Хамгийн сайн precision
        \item LightGBM (35\% жин) - Хурдан, үр ашигтай
        \item Random Forest (25\% жин) - Тогтвортой гүйцэтгэл
    \end{itemize}
    
    \item \textbf{BUY-Only стратеги:} SELL дохио 28\% accuracy-тай байсан тул хасаж, зөвхөн BUY дохиог ашигласан
    
    \item \textbf{70 техникийн индикатор:} 
    \begin{itemize}
        \item Trend индикаторууд (SMA, EMA, Golden Cross)
        \item Momentum индикаторууд (RSI, MACD, Stochastic)
        \item Volatility индикаторууд (ATR, Bollinger Bands)
        \item Candle patterns (Hammer, Engulfing)
        \item Support/Resistance (Pivot Points)
    \end{itemize}
\end{enumerate}

\subsection{Системийн бүрэлдэхүүн}

Бүрэн ажиллагаатай систем хөгжүүлсэн:

\begin{enumerate}
    \item \textbf{Backend API:} Flask + Waitress WSGI сервер
    \item \textbf{ML Pipeline:} Өгөгдөл татах → Индикатор тооцох → Ensemble таамаглал
    \item \textbf{Mobile App:} React Native + Expo мобайл аппликейшн
    \item \textbf{Database:} MongoDB (хэрэглэгчийн мэдээлэл, дохионууд)
    \item \textbf{Data Source:} Twelve Data API (бодит цагийн өгөгдөл)
\end{enumerate}

\section{Төгсгөлийн үг}

Энэхүү дипломын ажлаар XGBoost, LightGBM, Random Forest ensemble загварыг ашиглан EUR/USD валютын хосын BUY дохио таамаглах системийг амжилттай хөгжүүлсэн. 

80\%+ итгэлцэлтэй BUY сигнал 61.9\% win rate, 3.10 profit factor үзүүлж, backtest дээр +671 pip ашиг олсон. Энэ нь практикт хэрэглэгдэх боломжтой түвшин юм.

Машин сургалтаар санхүүгийн зах зээлийг төгс таамаглах боломжгүй боловч, статистик давуу талтай арилжааны шийдвэр дэмжлэгийн систем бүтээх бүрэн боломжтой гэдгийг энэхүү судалгаа харуулж байна.
