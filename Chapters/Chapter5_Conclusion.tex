\chapter{ДҮГНЭЛТ}

\section{Судалгааны үр дүнгийн нэгтгэл}

Энэхүү дипломын ажлаар Machine Learning Hybrid Ensemble аргыг ашиглан EUR/USD валютын хосын BUY дохио таамаглах системийг амжилттай хөгжүүлсэн. Судалгааны явцад XGBoost×3, LightGBM×2, CatBoost×2 гэсэн долоон загварыг нэгтгэсэн арга нь дангаар ажиллах загвараас маш их давуу, 96.9\% Accuracy үр дүн өгч байгааг баталсан.

\subsection{Техникийн шийдэл}

Загварын сургалтад ашигласан гол арга хэрэгслүүд:

\begin{enumerate}
    \item \textbf{Hybrid Ensemble загвар:} 7 загварын нэгдэл:
    \begin{itemize}
        \item XGBoost × 3 - Гурван өөр hyperparameter-тэй
        \item LightGBM × 2 - Histogram-based, санах ой хэмнэлттэй
        \item CatBoost × 2 - Ordered boosting, categorical feature сайн ажилладаг
    \end{itemize}
    
    \item \textbf{Agreement Bonus System:} Загваруудын зөвшилцөлд суурилсан нэмэлт оноо:
    \begin{itemize}
        \item 7/7 загвар ижил: +7\% bonus
        \item 6/7 загвар ижил: +4\% bonus
        \item 5/7 загвар ижил: +2\% bonus
    \end{itemize}
    
    \item \textbf{32 Technical Indicators:} 
    \begin{itemize}
        \item Trend Indicators (SMA, EMA, MACD, ADX)
        \item Momentum Indicators (RSI, Stochastic, CCI, Williams \%R)
        \item Volatility Indicators (ATR, Bollinger Bands)
        \item Time features (London/NY session, overlap)
    \end{itemize}
    
    \item \textbf{Confidence Filter:} 85\%+ confidence BUY дохио = 96.9\% accuracy
    
    \item \textbf{Dynamic SL/TP:} ATR дээр суурилсан Stop Loss, Take Profit тооцоолол
    
    \item \textbf{Entry/SL/TP Output:} Тодорхой entry price, stop loss, take profit утгууд
\end{enumerate}

\subsection{System Components}

Бүрэн ажиллагаатай систем хөгжүүлсэн:

\begin{enumerate}
    \item \textbf{Backend API:} Flask + Waitress WSGI Server
    \item \textbf{ML Pipeline:} Өгөгдөл татах → 32 Indicators тооцох → 7 загварын нэгдсэн таамаглал
    \item \textbf{Mobile App:} React Native + Expo мобайл аппликейшн
    \item \textbf{Database:} MongoDB (хэрэглэгчийн мэдээлэл, дохионууд)
    \item \textbf{Data Source:} Twelve Data API (бодит цагийн өгөгдөл)
\end{enumerate}

\section{Төгсгөлийн үг}

Энэхүү дипломын ажлаар Hybrid Ensemble загварыг (XGBoost×3, LightGBM×2, CatBoost×2) ашиглан EUR/USD валютын хосын BUY дохио таамаглах системийг амжилттай хөгжүүлсэн. 

85\%+ confidence BUY дохио нь backtest дээр \textbf{96.9\% Accuracy}, \textbf{46.5 Profit Factor} үзүүлж, ~55 өдрийн туршид \textbf{+910 пип} ашиг олсон. Энэ нь практикт маш сайн хэрэглэгдэх боломжтой түвшин юм.

7 загварын нэгдэл ба Agreement Bonus System-ийн хослол нь итгэлцүүрийг нэмэгдүүлж, алдааг маш бага түвшинд хүргэсэн (64 дохионоос зөвхөн 2 алдаатай). Machine Learning-ээр санхүүгийн зах зээлийг төгс таамаглах боломжгүй боловч, статистик давуу талтай арилжааны шийдвэр дэмжлэгийн систем бүтээх бүрэн боломжтой гэдгийг энэхүү судалгаа харуулж байна. 

Profit Factor 46.5 нь мэргэжлийн арилжаачдын хүлээн зөвшөөрдөг маш өндөр үр дүн бөгөөд цаашдын хөгжүүлэлтэд бат суурь болно.
